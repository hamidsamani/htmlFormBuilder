Any Element is Missing?

here you can learn how to add that element

Addition of element require two steps:

  • Adding element definition object
  • Adding element jade/pug template file

Adding element definition object

For defining element object, just define required attributes of element inside any related constant file. constant files placed in /js/constant folder

Every element should have this properties:

  • element [string]
  • attributes [array]
  • defaults [object]
  • vars [array]
  • required [array]
Also notice element root peoperty should be element name also (it's uses as template name also).

Adding element jade/pug template file

After creating the object definition take a look at jade user guide and create template file

template file should be placed in /app/htmlTemplates folder, notice that template file name should be equal to element name in element definition object.

also notice that if any element has related js script, put scripts in /app/jsTemplates folder. file name should be element name followed by -js. for example uiModal.pug script tag will be uiModal-js.pug
